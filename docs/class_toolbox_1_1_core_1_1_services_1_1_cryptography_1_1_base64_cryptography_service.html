Detailed Description

Cryptography service implementation using Base64 encoding.

This service provides text encryption and decryption using Base64 encoding. It supports both standard and URL-safe encoding tables.

Note: Base64 is an encoding scheme, not encryption. It provides obfuscation but not security. For secure encryption, use a proper encryption algorithm.

See also
ICryptographyService, Base64CryptographyOptions
