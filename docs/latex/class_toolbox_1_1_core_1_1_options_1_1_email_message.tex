\doxysubsection{Detailed Description}
Represents an email message to be sent. 

This class contains all the information needed to compose and send an email, including recipients, subject, body, and attachments. 

At least one recipient must be specified in \doxylink{class_toolbox_1_1_core_1_1_options_1_1_email_message_a2a22cf3d9bd7c67d8ac73aab098fa24e}{To} or \doxylink{class_toolbox_1_1_core_1_1_options_1_1_email_message_a1fde3f15e549c1ce10418ea24a8ee79b}{Bcc}. If \doxylink{class_toolbox_1_1_core_1_1_options_1_1_email_message_a2a22cf3d9bd7c67d8ac73aab098fa24e}{To} is empty, at least one \doxylink{class_toolbox_1_1_core_1_1_options_1_1_email_message_a1fde3f15e549c1ce10418ea24a8ee79b}{Bcc} recipient is required.
