\doxysubsection{Detailed Description}
Abstract base class providing a thread-\/safe implementation of the dispose pattern with integrated telemetry support. 

This class implements \doxylink{interface_toolbox_1_1_core_1_1_abstractions_1_1_i_disposable_service}{IDisposable\+Service} and \doxylink{interface_toolbox_1_1_core_1_1_abstractions_1_1_i_instrumented_service}{IInstrumented\+Service}, providing a consistent foundation for building disposable services. 

Derived classes should override \doxylink{class_toolbox_1_1_core_1_1_base_1_1_base_disposable_service_abe66f7a1e36466851f1a3df7ce2cc51c}{Dispose(bool)} to release their resources. The base implementation handles thread-\/safety and telemetry automatically.
