\doxysubsection{Detailed Description}
Defines the contract for HTTP API services. 

This interface provides methods for making HTTP requests with support for:\+ 


\begin{DoxyItemize}
\item All HTTP verbs (GET, POST, PUT, DELETE, PATCH, etc.) 
\item Multiple authentication modes (Bearer, Basic, API Key, Certificate) 
\item JSON serialization/\+deserialization 
\item Custom headers and query parameters 
\item Automatic retry with exponential backoff 
\end{DoxyItemize}
