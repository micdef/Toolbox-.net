\doxysubsection{Detailed Description}
HTTP API service implementation using Http\+Client. 

This service provides HTTP request capabilities with support for:\+ 


\begin{DoxyItemize}
\item All HTTP verbs 
\item Multiple authentication modes 
\item Automatic retry with exponential backoff 
\item Request/\+response logging and telemetry 
\end{DoxyItemize}
