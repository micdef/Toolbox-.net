\doxysubsection{Detailed Description}
Interface for services that expose Open\+Telemetry instrumentation. 

\doxylink{namespace_toolbox_1_1_core_1_1_abstractions_1_1_services}{Services} implementing this interface provide access to their telemetry primitives, enabling consistent observability across the application.
