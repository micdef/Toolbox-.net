\doxysubsection{Detailed Description}
Configuration options for the SMTP mailing service. 

These options configure how the mailing service connects to an SMTP server. Authentication is optional for servers that support anonymous relay. 

For secure connections, use \doxylink{namespace_toolbox_1_1_core_1_1_options_a0a1f76d9ade3c05f751b22ef05061453a645e26a3aaab46321ec5ea64e6ebcbb1}{Smtp\+Security\+Mode.\+Start\+Tls} (port 587) or \doxylink{namespace_toolbox_1_1_core_1_1_options_a0a1f76d9ade3c05f751b22ef05061453a1ae39b3b0584084d66a8b31f7f43b6b1}{Smtp\+Security\+Mode.\+Ssl\+On\+Connect} (port 465).
