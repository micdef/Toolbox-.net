\doxysubsection{Detailed Description}
\doxylink{namespace_toolbox_1_1_core_1_1_services_1_1_cryptography}{Cryptography} service implementation using AES symmetric encryption. 

This service provides secure text encryption and decryption using the AES algorithm. It uses CBC mode with PKCS7 padding. 

The encrypted output is Base64-\/encoded for safe text transmission.
