\doxysubsection{Detailed Description}
Interface for services requiring asynchronous disposal with cancellation support. 

This interface combines \doxylink{interface_toolbox_1_1_core_1_1_abstractions_1_1_i_disposable_service}{IDisposable\+Service} and IAsync\+Disposable with an additional method supporting cancellation tokens for graceful shutdown scenarios.
