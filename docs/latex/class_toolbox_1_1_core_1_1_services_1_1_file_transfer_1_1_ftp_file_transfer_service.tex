\doxysubsection{Detailed Description}
File transfer service implementation using FTP/\+\+FTPS protocols. 

This service provides file upload and download capabilities using the FTP protocol. It supports both plain FTP and FTPS (FTP over TLS/\+\+SSL). 

Uses Fluent\+FTP library for FTP operations.
