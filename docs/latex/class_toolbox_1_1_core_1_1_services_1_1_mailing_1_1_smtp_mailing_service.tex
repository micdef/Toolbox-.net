\doxysubsection{Detailed Description}
Email sending service implementation using SMTP with Mail\+Kit. 

This service provides secure email sending via SMTP with support for:\+ 


\begin{DoxyItemize}
\item TLS/\+\+SSL encryption (STARTTLS and implicit SSL) 
\item Password and OAuth2 authentication 
\item Anonymous connections for internal relays 
\item Multiple recipients (To, CC, BCC) 
\item HTML and plain text content 
\item File attachments and inline images 
\end{DoxyItemize}
