\doxysubsection{Detailed Description}
Interface for cryptography services providing text encryption and decryption. 

This interface defines a simple contract for encrypting and decrypting text. Implementations may use various algorithms (Base64, AES, RSA, etc.). 

All methods accept and return string values for ease of use.
