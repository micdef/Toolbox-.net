\doxysubsection{Detailed Description}
Defines the contract for email sending services. 

This interface provides methods for sending emails with support for:\+ 


\begin{DoxyItemize}
\item Single or multiple recipients (To, CC, BCC) 
\item File attachments 
\item HTML or plain text content 
\end{DoxyItemize}
