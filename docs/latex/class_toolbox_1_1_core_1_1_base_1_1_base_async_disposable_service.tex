\doxysubsection{Detailed Description}
Abstract base class providing asynchronous disposal with cancellation support and integrated telemetry. 

This class extends \doxylink{class_toolbox_1_1_core_1_1_base_1_1_base_disposable_service}{Base\+Disposable\+Service} to implement \doxylink{interface_toolbox_1_1_core_1_1_abstractions_1_1_i_async_disposable_service}{IAsync\+Disposable\+Service}, enabling graceful async cleanup with timeout support through cancellation tokens. 

Derived classes should override \doxylink{class_toolbox_1_1_core_1_1_base_1_1_base_async_disposable_service_af67f7795ee7795cbe84c409e7c48af62}{Dispose\+Async\+Core(\+Cancellation\+Token)} to release their async resources.
