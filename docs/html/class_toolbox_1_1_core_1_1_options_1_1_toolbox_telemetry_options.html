Detailed Description

Configuration options for OpenTelemetry integration in Toolbox.

These options can be configured via appsettings.json:

{
"Toolbox:Telemetry": {
"EnableTracing": true,
"EnableMetrics": true,
"OtlpEndpoint": "http://localhost:4317"
}
}
