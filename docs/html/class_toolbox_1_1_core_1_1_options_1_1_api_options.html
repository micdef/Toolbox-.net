Detailed Description

Configuration options for the HTTP API service.

These options configure HTTP client behavior and authentication. The authentication mode determines which credential properties are used.

var options = new ApiOptions
{
BaseUrl = "https://api.example.com",
AuthenticationMode = ApiAuthenticationMode.BearerToken,
BearerToken = "your-token-here",
Timeout = TimeSpan.FromSeconds(30)
};
