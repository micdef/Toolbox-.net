Detailed Description

Base interface for disposable services providing lifecycle management.

This interface extends IDisposable with additional properties for service identification and disposal state tracking.

Gets a value indicating whether this service has been disposed.

true if the service has been disposed; otherwise, false.

This property should be checked before performing any operation on the service. Operations on disposed services should throw ObjectDisposedException.
