Detailed Description

Configuration options for the SMTP mailing service.

These options configure how the mailing service connects to an SMTP server. Authentication is optional for servers that support anonymous relay.

For secure connections, use SmtpSecurityMode.StartTls (port 587) or SmtpSecurityMode.SslOnConnect (port 465).

var options = new MailingOptions
{
Host = "smtp.example.com",
Port = 587,
SecurityMode = SmtpSecurityMode.StartTls,
Username = "user@example.com",
Password = "password"
};
