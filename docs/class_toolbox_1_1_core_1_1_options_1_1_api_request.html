Detailed Description

Represents an HTTP API request.

This class encapsulates all the data needed to make an HTTP request, including URL, method, headers, and body content.

// GET request
var request = new ApiRequest("GET", "/users");
// POST with JSON body
var request = ApiRequest.Post("/users", new { Name = "John" });
// With custom headers
var request = new ApiRequest("GET", "/data")
.WithHeader("X-Custom", "value");
